# Home-Link Help

Welcome to Home-Link! This is a personalized dashboard to organize your favorite links.

## Features

- **Link Management**: Add, edit, and delete links.
- **Customization**: Change the theme and appearance.
- **User Roles**: Switch between admin and user views to manage the dashboard.
